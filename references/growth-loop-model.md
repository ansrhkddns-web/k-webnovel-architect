# 성장 루프 모델

한국 웹소설의 성장 보상은 숫자, 랭크, 돈, 명성, 관계, 단서, 영토, 기술 숙련도처럼 여러 형태로 나타난다. 중요한 것은 보상이 `정보`로 끝나지 않고, 주인공의 행동과 주변 반응을 바꾸는 것이다.

## 기본 루프

```text
압박 또는 문제
-> 주인공의 선택
-> 행동과 실패/대가
-> 보상 또는 기록
-> 체감 변화
-> 주변 반응
-> 다음 욕망 또는 다음 위험
```

## 루프 필드

| 필드 | 질문 |
|---|---|
| 압박 | 주인공이 움직이지 않으면 무엇을 잃는가? |
| 선택 | 주인공이 어떤 비용 있는 결정을 하는가? |
| 행동 | 실제 장면에서 무엇을 하는가? |
| 실패/대가 | 보상을 얻기 전에 어떤 손해, 공포, 오해, 부상을 겪는가? |
| 보상 | 무엇이 오른다, 생긴다, 드러난다, 바뀌는가? |
| 체감 | 독자가 그 변화를 어떤 장면으로 확인하는가? |
| 주변 반응 | 누가 알아차리고, 의심하고, 질투하고, 인정하는가? |
| 다음 압력 | 보상 때문에 어떤 새 문제가 생기는가? |

## 보상 종류

| 보상 | 쓰기 좋은 장르 | 체감 방식 |
|---|---|---|
| 신체/전투 성장 | 헌터, 무협, 아카데미, 탑 | 이전에 못 하던 행동을 한다 |
| 스킬/능력 해금 | 시스템, 게임, 탑, 헌터 | 선택지가 늘어나지만 조건도 생긴다 |
| 돈/자산 | 재벌, 사업, 전문직, 생존물 | 더 큰 판에 들어가고 더 큰 적을 만난다 |
| 지위/명성 | 아카데미, 로판, 직업물, 아이돌 | 주변 태도와 접근 권한이 바뀐다 |
| 관계 변화 | 로맨스, 로판, 가족물, 팀물 | 신뢰, 의심, 집착, 질투가 움직인다 |
| 단서/정보 | 미스터리, 괴담, 정치물 | 현재 문제 해결에 작게 도움을 준다 |
| 영토/조직 | 영지물, 생존물, 국가건설물 | 운영 문제와 책임이 늘어난다 |
| 감정 해방 | 복수, 학원, 후회물 | 억울함이 해소되지만 새 후폭풍이 온다 |

## 숫자 보상 규칙

숫자, 랭크, 포인트, 호감도, 재산, 조회수 같은 수치는 반드시 체감 장면이 필요하다.

좋은 처리:
- 수치가 오른다.
- 주인공이 아주 작은 변화를 확인한다.
- 그 변화가 욕망이나 두려움을 만든다.
- 누군가 그 변화를 목격하거나 의심한다.

나쁜 처리:
- 수치만 길게 나열한다.
- 수치 상승이 다음 행동을 바꾸지 않는다.
- 수치가 너무 크게 뛰어 긴장이 무너진다.
- 보상이 반복되는데 장면의 의미는 같다.

## 체감 장면 설계

보상은 아래 다섯 통로 중 하나로 체감시키면 좋다.

| 통로 | 설명 | 사용 예 |
|---|---|---|
| 몸 | 움직임, 통증, 피로, 회복, 감각이 바뀐다 | 이전에 못 버티던 것을 버틴다 |
| 물건 | 도구, 돈, 아이템, 기록, 계약서가 남는다 | 손에 잡히는 증거가 생긴다 |
| 관계 | 누군가 태도를 바꾼다 | 무시하던 사람이 의심하거나 인정한다 |
| 제도 | 랭크, 시험, 계약, 법, 조직 반응이 바뀐다 | 접근 권한이나 책임이 늘어난다 |
| 선택지 | 주인공이 할 수 있는 일이 늘어난다 | 도망만 치던 인물이 협상/반격을 고른다 |

체감 장면은 작아도 된다. 중요한 것은 독자가 “이전과 다르다”를 느끼는 것이다.

## 성장 단계

| 단계 | 기능 | 주의점 |
|---|---|---|
| 발견 | 비정상적인 보상 구조를 처음 본다 | 설명보다 혼란과 확인 욕구를 먼저 둔다 |
| 실험 | 주인공이 규칙을 직접 시험한다 | 실험은 일상 물건, 작은 관계, 작은 위기로 체감시킨다 |
| 첫 활용 | 보상을 실제 문제 해결에 쓴다 | 너무 큰 승리보다 작은 역전이 좋다 |
| 대가 | 성장 방식의 비용이 드러난다 | 힘이 생긴 만큼 의심, 책임, 적대도 커져야 한다 |
| 숙련 | 주인공이 자기 방식으로 규칙을 다룬다 | 반복 사냥이나 반복 훈련만으로 늘어지지 않게 한다 |
| 확장 | 개인 성장이 사회적 지위나 세계 문제로 커진다 | 초반 장르 약속을 버리지 말고 확장한다 |

## 실패와 대가

성장이 재미있으려면 실패와 대가가 있어야 한다. 대가는 주인공을 약하게 만들기 위한 장치가 아니라, 다음 회차의 압력을 만드는 장치다.

| 대가 | 효과 | 주의 |
|---|---|---|
| 신체 대가 | 부상, 피로, 과부하로 긴장 유지 | 매번 쓰면 답답해진다 |
| 사회 대가 | 의심, 질투, 소문, 추적 발생 | 억지 오해로 끌지 않는다 |
| 자원 대가 | 돈, 시간, 아이템, 기회 소모 | 회복 방법도 설계한다 |
| 감정 대가 | 죄책감, 공포, 집착, 외로움 | 감상문이 아니라 선택 변화로 보여준다 |
| 정보 대가 | 알수록 더 위험해짐 | 답을 너무 늦게 주지 않는다 |
| 도덕 대가 | 이기기 위해 선을 넘을지 고민 | 장르 톤과 맞춰야 한다 |

## 반복 피로 방지

같은 성장 루프를 반복할 때는 아래 중 하나를 바꾼다.

- 장소를 바꾼다.
- 상대를 바꾼다.
- 보상 종류를 바꾼다.
- 실패 이유를 바꾼다.
- 주변 반응을 바꾼다.
- 주인공의 목표를 바꾼다.
- 같은 행동의 효율을 떨어뜨린다.
- 보상이 새로운 위험을 부르게 한다.

## 장르별 성장 루프

| 장르 | 성장 단위 | 반복 루프 |
|---|---|---|
| 헌터/게이트 | 능력, 랭크, 아이템, 공략 정보 | 위기 -> 공략 -> 보상 -> 인정/의심 -> 더 큰 던전 |
| 회귀 | 미래 지식, 자산, 관계 선점 | 과거 문제 -> 선점 -> 변수 발생 -> 수정 전략 |
| 빙의/악역 | 생존 플래그, 평판, 관계 | 원작 위험 -> 회피/역이용 -> 관계 변화 -> 새 플래그 |
| 무협 | 내공, 무공, 명분, 사문 지위 | 굴욕/위협 -> 수련/결투 -> 체면 회복 -> 더 큰 문파 갈등 |
| 로맨스판타지 | 평판, 권력, 계약, 감정 | 사회적 위기 -> 선택 -> 관계 균열/진전 -> 새 스캔들 |
| 직업물 | 실적, 전문성, 계약, 고객 신뢰 | 문제 의뢰 -> 전문 판단 -> 결과 -> 업계 반응 |
| 스포츠/아이돌 | 기록, 팬덤, 무대, 팀 신뢰 | 훈련/무대 -> 성과 -> 여론 변화 -> 더 큰 경기/무대 |
| 미스터리/괴담 | 단서, 생존 규칙, 금기 이해 | 이상 현상 -> 조사 -> 부분 답 -> 더 위험한 질문 |

## 이탈 방지

- 보상이 늦으면 중간에 작은 체감 보상을 넣는다.
- 주인공이 같은 방식으로만 이기면 장애물의 종류를 바꾼다.
- 설명이 길어지면 보상 규칙을 장면 안의 실험으로 보여준다.
- 성장 때문에 생긴 후폭풍을 반드시 만든다.
- 독자가 기다리는 큰 사이다 전에 작은 사이다를 배치한다.
