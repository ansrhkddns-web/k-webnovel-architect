# 원고/기획 리뷰 루브릭

기존 원고나 설정집을 리뷰할 때는 감상보다 `상업 연재 기능`을 먼저 본다. 이 문서는 리뷰 점수와 구체적 수정 지시를 만들기 위한 기준이다.

## 리뷰 항목

| 항목 | 질문 | 점수 |
|---|---|---|
| 장르 약속 | 제목/소개/1화가 같은 장르를 말하는가? | 1-5 |
| 주인공 엔진 | 주인공 욕망과 선택이 회차를 미는가? | 1-5 |
| 초반 훅 | 첫 화면과 1화 엔딩이 강한가? | 1-5 |
| 반복 보상 | 2-4화마다 보상이 있는가? | 1-5 |
| 체감성 | 변화가 장면과 주변 반응으로 보이는가? | 1-5 |
| 적대자 기능 | 적이 주인공 성장을 검증하는가? | 1-5 |
| 인물망 | 인물 기능이 겹치지 않는가? | 1-5 |
| 설명 소화 | 설정이 사건 안에서 제시되는가? | 1-5 |
| 떡밥 관리 | 단기 답과 장기 질문이 균형 잡혔는가? | 1-5 |
| 유료화 신뢰 | 무료분에서 결제할 이유를 만들었는가? | 1-5 |

## 리뷰 출력 순서

1. 치명 문제.
2. 상업성 강점.
3. 구간별 이탈 위험.
4. 수정 우선순위.
5. 다음 작업 지시.

칭찬을 먼저 늘어놓지 않는다. 사용자가 바로 고칠 수 있도록 문제와 처방을 함께 준다.

```markdown

-

1.
```

## 치명 판정

아래는 우선 수정한다.

- 제목과 본문 장르 약속이 다르다.
- 1화에 장르 핵심 소재가 안 보인다.
- 주인공이 5화까지 선택하지 않는다.
- 보상이 5화까지 없다.
- 적대자가 기능 없이 불쾌감만 준다.
- 설정 설명이 초반 장면을 압도한다.
- 유료화 직전까지 신뢰 보상이 없다.
