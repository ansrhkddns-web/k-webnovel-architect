# 1화 훅 라이브러리

1화 훅은 독자를 놀라게 하는 장치가 아니라, 제목과 소개글의 약속을 장면으로 증명하는 장치다. 좋은 훅은 `압박`, `비정상성`, `선택`, `다음 질문`을 빠르게 만든다.

## 훅 원칙

- 첫 화면에는 주인공의 현재 압박 또는 비정상 사건이 있어야 한다.
- 1화 안에 장르 핵심 소재가 등장하거나 강하게 예고되어야 한다.
- 마지막은 추상적인 불길함보다 구체적인 질문이 좋다.
- 훅은 장기 떡밥보다 단기 클릭 이유가 먼저다.
- 2화에서 바로 확인할 장면을 약속해야 한다.

## 훅 유형

| 유형 | 구조 | 잘 맞는 장르 | 위험 |
|---|---|---|---|
| 즉시 위기형 | 위험이 이미 닥친 상태에서 시작 | 헌터, 공포, 스릴러, 아포칼립스 | 주인공 처지 설명 부족 |
| 굴욕/압박형 | 주인공의 낮은 처지를 장면으로 보여줌 | 성장, 복수, 학원, 무협 | 고구마가 길어짐 |
| 비정상 발견형 | 일상에 없던 것이 생김 | 현대판타지, 괴담, 던전독점 | 발견만 하고 행동이 늦음 |
| 죽음/실패형 | 파멸 직전 또는 사망 후 기회 | 회귀, 복수, 로판 | 과거 설명 과밀 |
| 역할 각성형 | 빙의/환생/원작 역할 인식 | 빙의, 악녀, 아카데미 | 원작 설명이 길어짐 |
| 공개 평가형 | 시험, 무대, 경기, 결투 | 아카데미, 스포츠, 아이돌, 무협 | 결과가 늦으면 지루함 |
| 계약/스캔들형 | 관계나 지위가 흔들리는 사건 | 로맨스, 로판, 오피스 | 감정 압력 부족 |
| 사건 의뢰형 | 해결해야 할 사건이 들어옴 | 전문직, 추리, 법정, 의학 | 주인공 개인 욕망 부족 |
| 금기 위반형 | 하면 안 되는 행동을 하게 됨 | 공포, 오컬트, 데스게임 | 규칙이 임의적으로 보임 |
| 귀환/복귀형 | 강자가 낮은 무대로 돌아옴 | 귀환자, 무협, 헌터 | 너무 쉽게 이김 |

## 장르별 1화 필수 확인

| 장르 | 1화에서 반드시 보여줄 것 |
|---|---|
| 헌터/게이트 | 위험한 초현실 사건과 주인공의 비정상 반응 |
| 회귀 | 실패한 과거의 무게와 새 선택지 |
| 빙의/악녀 | 역할의 파멸 위험과 첫 이탈 행동 |
| 무협 | 굴욕/은원/무공 정체성 중 하나 |
| 로판 | 사회적 위험과 관계/지위 압박 |
| 아카데미 | 경쟁 규칙과 주인공의 낮거나 이상한 위치 |
| 전문직 | 현장 문제와 주인공 판단의 차이 |
| 스포츠/아이돌 | 공개 검증 무대와 욕망 |
| 공포/괴담 | 이상 현상과 금기 |
| 아포칼립스 | 재난의 시작과 생존 선택 |

## 첫 화면 점검

```markdown
## 1화 첫 화면 점검
- 첫 문단의 압박:
- 첫 장면의 장소:
- 주인공이 잃을 수 있는 것:
- 독자가 보는 장르 신호:
- 불필요한 설명:
- 2화로 넘기는 질문:
```

## 1화 엔딩 훅

좋은 엔딩 훅:
- 누군가 주인공의 변화를 봤다.
- 숫자, 계약, 증거, 메시지, 문서처럼 구체물이 남았다.
- 적이 다음 행동을 시작했다.
- 보상을 받았지만 더 큰 비용이 드러났다.
- 이상 현상이 현실로 새어나왔다.
- 관계가 되돌릴 수 없게 틀어졌다.
- 첫 승리 직후 더 큰 판이 보였다.

약한 엔딩 훅:
- “뭔가 시작됐다.”
- “나는 몰랐다.”
- “불길했다.”
- “그때는 알지 못했다.”
- “세상이 변하고 있었다.”

약한 훅은 구체물, 구체 행동, 구체 위험으로 바꾼다.

## 1화 리라이트 처방

| 문제 | 처방 |
|---|---|
| 세계 설명이 먼저 온다 | 설명을 삭제하고 압박 장면으로 시작한다 |
| 장르 소재가 늦다 | 핵심 소재를 1화 중반 이전으로 당긴다 |
| 주인공이 보기만 한다 | 작은 선택, 실수, 도망, 거절 중 하나를 시킨다 |
| 엔딩이 흐릿하다 | 2화에서 바로 확인할 물건/상대/위협을 남긴다 |
| 고구마가 강하다 | 아주 작은 반격이나 정보 보상을 넣는다 |
| 특수성이 안 보인다 | 남들과 다른 반응 또는 결과를 넣는다 |
