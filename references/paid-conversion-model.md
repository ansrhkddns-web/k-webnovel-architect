# 유료화 전환 모델

유료화는 이야기를 끊는 위치가 아니라, 독자가 이미 받은 보상을 믿고 다음 보상에 돈을 내는 위치다. 좋은 유료화 지점은 `직전 보상`과 `다음 약속`이 동시에 있어야 한다.

## 기본 공식

```text
압박 누적
-> 작은 보상 여러 번
-> 중간 사이다 또는 큰 변화
-> 더 큰 판의 문이 열림
-> 유료화
-> 유료분 초반에 약속한 보상 회수
```

## 좋은 유료화 후보

| 후보 | 직전 보상 | 다음 약속 |
|---|---|---|
| 첫 주요 사이다 직후 | 억울함 일부 해소 | 더 큰 적/조직 등장 |
| 첫 공개 인정 직후 | 랭크, 성적, 평판 상승 | 더 큰 무대 진입 |
| 첫 관계 전환 직후 | 계약/고백/동맹/배신 | 사회적 후폭풍 |
| 첫 큰 공략 성공 직후 | 전투/시험/의뢰 성공 | 숨은 규칙 노출 |
| 첫 비밀 노출 직후 | 주인공 변화가 목격됨 | 추적/질투/보호 시작 |
| 첫 세계 확장 직후 | 개인 문제 너머의 진실 | 새 세력/재난 |

## 나쁜 유료화 후보

| 후보 | 문제 | 보정 |
|---|---|---|
| 보상 전 절단 | 독자는 결제 신뢰가 없다 | 직전 작은 사이다를 넣는다 |
| 설명 직후 절단 | 감정 에너지가 낮다 | 장면 변화 뒤에 둔다 |
| 주인공 패배 직후 | 돈 내고 답답함을 사는 느낌 | 패배 속 정보 보상을 준다 |
| 장르 전환 직전 | 무료분 약속과 유료분 약속이 다르다 | 전환 씨앗을 미리 심는다 |
| 떡밥만 추가 | 답 없이 질문만 늘어난다 | 부분 답을 먼저 준다 |

## 무료분 구간별 목표

| 구간 | 목표 | 최소 보상 |
|---|---|---|
| 1화 | 클릭 유지 | 장르 신호, 압박, 구체 훅 |
| 2-5화 | 반복 재미 증명 | 첫 사용, 첫 보상 |
| 6-10화 | 신뢰 형성 | 보상 체감, 대가, 적대 강화 |
| 11-15화 | 중간 추진 | 관계/지위/정보 변화 |
| 16-25화 | 결제 전 신뢰 | 중간 사이다 또는 큰 변화 |
| 유료화 직후 | 결제 보상 | 직전 훅의 빠른 회수 |

## 장르별 유료화 훅

| 장르 | 좋은 훅 |
|---|---|
| 헌터 | 첫 큰 공략 성공 후 상위 던전/기관 개입 |
| 회귀/재벌 | 첫 선점 성공 후 예상 밖 변수 |
| 무협 | 공개 결투 승리 후 문파/세력 충돌 |
| 아카데미 | 첫 평가 뒤 파벌 또는 상급 시험 |
| 로판 | 첫 사교/계약 승리 뒤 더 큰 스캔들 |
| 로맨스/BL/GL | 관계 정의가 흔들린 뒤 외부 압박 |
| 스포츠/아이돌 | 첫 무대/경기 성공 뒤 더 큰 공개 평가 |
| 전문직 | 첫 사건 해결 뒤 더 큰 의뢰/사회적 사건 |
| 공포/미스터리 | 작은 규칙 해명 뒤 더 큰 금기 |
| 아포칼립스 | 첫 안전 확보 뒤 더 큰 집단/재난 |

## 유료분 초반 보상

유료 전환 직후 1-3화 안에 직전 훅을 회수하거나, 최소한 보상 방향을 보여준다.

나쁜 처리:
- 유료화 직후 새 설명을 길게 시작한다.
- 직전 훅을 방치하고 다른 사건으로 간다.
- 주인공이 다시 아무것도 못 하는 상태로 돌아간다.

좋은 처리:
- 직전 훅에 바로 반응한다.
- 유료화 직전 보상이 후폭풍을 만든다.
- 새 무대에 들어가되 기존 보상 루프를 유지한다.
