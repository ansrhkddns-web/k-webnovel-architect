# 회차 브리프 생성기

회차 브리프는 원고를 쓰기 전 작성하는 짧은 지시서다. 줄거리 요약이 아니라, 이번 화가 상업 연재에서 어떤 기능을 수행해야 하는지 정한다.

## 회차 브리프 필드

```markdown
## N화 브리프

### 회차 기능
-

### 시작 압박
-

### 주인공 목표
-

### 필수 장면
1.
2.
3.

### 독자 보상
-

### 체감 장면
-

### 주변 반응
-

### 심거나 회수할 떡밥
-

### 금지할 것
-

### 엔딩 훅
-

### 다음 화 연결
-
```

## 회차 기능 목록

| 기능 | 설명 |
|---|---|
| 발견 | 핵심 소재나 규칙을 처음 본다 |
| 재시도 | 주인공이 다시 핵심 사건에 손댄다 |
| 실험 | 보상/규칙을 확인한다 |
| 첫 보상 | 독자가 계속 볼 근거를 얻는다 |
| 외부 반응 | 누군가 변화를 목격한다 |
| 적대 강화 | 적이 더 강하게 압박한다 |
| 관계 전환 | 인물 관계의 의미가 바뀐다 |
| 중간 사이다 | 작은 억울함이 풀린다 |
| 대가 노출 | 능력/선택의 비용이 드러난다 |
| 세계 확장 | 더 큰 무대나 세력이 열린다 |
| 부분 회수 | 떡밥 일부가 답을 준다 |

## 브리프 작성 규칙

- 이번 화의 보상을 하나만 명확히 정한다.
- 금지할 설명을 미리 적는다.
- 엔딩 훅은 구체물, 행동, 위협, 선택 중 하나로 만든다.
- 전 화의 훅에 반응하지 않고 새 사건으로 도망가지 않는다.
- 큰 설정 설명은 현재 장면의 문제 해결과 연결될 때만 허용한다.

## 장르별 회차 보상

| 장르 | 좋은 회차 보상 |
|---|---|
| 헌터/탑 | 공략 정보, 아이템, 랭크 반응, 새 위기 |
| 회귀/재벌 | 선점, 계약, 변수, 평판 변화 |
| 로판/로맨스 | 관계 거리 변화, 평판, 계약, 스캔들 |
| 무협 | 초식, 경지, 체면, 은원 변화 |
| 전문직 | 사건 해결, 고객/기관 반응, 더 큰 의뢰 |
| 아이돌/스포츠 | 무대/경기 반응, 기록, 팀 신뢰 |
| 공포/미스터리 | 작은 규칙, 단서, 생존, 더 큰 질문 |

## 브리프 검수

- 이번 화를 한 문장으로 설명할 수 있는가?
- 끝났을 때 상황이 달라지는가?
- 독자가 받은 보상이 보이는가?
- 다음 화를 누를 이유가 구체적인가?
- 기존 장르 약속과 맞는가?
