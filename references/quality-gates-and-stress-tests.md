# 품질 게이트와 스트레스 테스트

기획이 좋아 보여도 실제 연재에서 무너지는 경우가 많다. 이 문서는 상업 연재 전에 통과해야 할 품질 게이트와 스트레스 테스트를 제공한다.

## 10초 테스트

아래 질문에 10초 안에 답할 수 있어야 한다.

- 무슨 장르인가?
- 주인공은 어떤 처지인가?
- 무엇이 특별한가?
- 독자는 무엇을 반복해서 받는가?
- 1화 끝에서 무엇이 궁금한가?

답이 길어지면 콘셉트가 흐린 것이다.

## 5화 테스트

5화까지 아래가 보여야 한다.

- 핵심 소재를 실제로 사용했다.
- 첫 작은 보상을 줬다.
- 주인공이 선택으로 사건을 움직였다.
- 반복 문제의 원천이 보인다.
- 더 큰 세계나 적의 씨앗이 있다.

## 25화 테스트

무료분 또는 첫 유료화 후보 전까지 아래가 필요하다.

- 중간 사이다 또는 큰 관계 변화.
- 주인공 처지의 명확한 변화.
- 초반 떡밥 하나의 부분 회수.
- 더 큰 적이나 무대의 개방.
- 유료분 초반에 줄 보상의 약속.

## 50화 테스트

장기 연재 가능성을 확인한다.

- 초반 보상이 사라지지 않고 확장되었는가?
- 적대자 단계가 커졌는가?
- 주인공 목표가 새롭게 정의되었는가?
- 세계관 규칙이 사건을 계속 만드는가?
- 반복 피로를 줄일 변주가 있는가?

## 스트레스 테스트

| 테스트 | 질문 | 실패 시 보정 |
|---|---|---|
| 제목 제거 테스트 | 제목 없이 1화만 봐도 장르가 보이는가? | 1화 초반에 장르 신호 추가 |
| 1화 제거 테스트 | 소개글만 봐도 클릭 이유가 있는가? | 제목/소개글 약속 강화 |
| 보상 제거 테스트 | 초반 보상 하나를 빼도 유지되는가? | 보상 간격 재배치 |
| 적대자 교체 테스트 | 적대자를 바꿔도 같은 장면이면 기능이 약한가? | 적대자별 압박 방식 분리 |
| 주인공 욕망 테스트 | 주인공 욕망을 한 문장으로 말할 수 있는가? | 외적 목표와 결핍 재설계 |
| 장르 혼합 테스트 | 보조 장르를 빼도 주 장르가 작동하는가? | 주 장르 계약 재정의 |
| 100화 테스트 | 100화쯤 무엇이 커져 있는가? | 장기 사다리 추가 |
| 유료화 신뢰 테스트 | 돈 내기 전 이미 보상을 받았는가? | 무료분 중간 보상 추가 |

## 위험 등급

| 등급 | 의미 | 대응 |
|---|---|---|
| 낮음 | 보강하면 충분 | 회차표에서 수정 |
| 중간 | 초반부 구조 수정 필요 | 1-10화 재배치 |
| 높음 | 장르 계약/주인공 엔진 흔들림 | 콘셉트부터 재설계 |
| 치명 | 독자 약속과 실제 내용이 다름 | 제목/소개/장르 또는 본문 방향 중 하나를 바꾼다 |
