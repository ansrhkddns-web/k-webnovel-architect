# 데이터 모델

한국 웹소설 작품 기획용 내부 데이터 모델.

상위 판매권 500개 작품군을 기준 표본으로 삼고, 공개적으로 확인 가능한 랭킹, 장르 분포, 조회수, 유료 전환, 장기 연재, 웹툰화, 영상화, 게임화, 팬덤 확장 신호에서 반복되는 설계 데이터를 축적한다.

특정 작품명, 줄거리 예시, 고유 사건은 기록하지 않는다. 사용자는 결과물에서 필요한 항목만 본다.
