# 연재 전략

연재 전략은 작품을 시장에 맞춰 납작하게 만드는 작업이 아니다. 독자가 어떤 약속을 보고 들어오고, 어느 지점에서 신뢰하고, 언제 결제를 고민하는지 설계하는 작업이다.

## 패키징 체인

```text
제목
-> 태그
-> 소개글
-> 1화 첫 화면
-> 1화 엔딩
-> 5화까지의 반복 보상
-> 무료분 끝의 큰 약속
-> 유료분 초반의 보상 유지
```

이 체인이 같은 장르 약속을 말해야 한다. 제목은 헌터물인데 소개글은 감성 성장물, 1화는 가족 드라마라면 독자는 속았다고 느낄 수 있다.

플랫폼 성격별 세부 패키징은 `platform-packaging-model.md`를 함께 본다. 유료화 후보는 `paid-conversion-model.md`, IP 확장은 `ip-adaptation-model.md`, 품질 검증은 `quality-gates-and-stress-tests.md`를 함께 본다.

## 제목 전략

제목은 보통 아래 기능 중 두 가지 이상을 해야 한다.

- 장르를 즉시 보여준다.
- 주인공의 처지나 우위를 보여준다.
- 반복 보상을 암시한다.
- 차별화 포인트를 남긴다.
- 50화 이후에도 어색하지 않다.

진단:
```markdown
| 제목 | 장르 명료도 | 클릭 욕구 | 보상 암시 | 차별성 | 장기 적합성 | 코멘트 |
|---|---:|---:|---:|---:|---:|---|
|  |  |  |  |  |  |  |
```

## 소개글 구조

소개글은 문학적 요약보다 구매 약속에 가깝다.

권장 구조:
1. 주인공의 처지.
2. 비정상 사건 또는 특수한 우위.
3. 반복 보상.
4. 장기 확장 또는 더 큰 위기.

```markdown
[주인공 처지]였던 주인공은 [비정상 사건/우위]를 얻는다.
처음에는 [작은 목표]를 위해 움직이지만,
곧 [더 큰 세계/적/재난/관계]와 맞닥뜨린다.
이 작품의 재미는 [반복 보상]이다.
```

피한다:
- 세계관 고유명사를 첫 문장부터 나열한다.
- 분위기만 말하고 주인공이 무엇을 하는지 말하지 않는다.
- 장르 핵심 보상을 숨긴다.
- 결말급 떡밥을 소개글에서 과하게 설명한다.

## 태그 설계

태그는 독자 유입 경로다. 멋진 단어보다 검색과 기대에 맞는 단어가 우선이다.

태그 묶음:
- 장르 태그: 헌터, 회귀, 빙의, 아카데미, 무협, 로판, 스포츠 등.
- 보상 태그: 성장, 사이다, 복수, 먼치킨, 생존, 경영, 힐링 등.
- 장치 태그: 시스템, 상태창, 게이트, 탑, 던전, 계약, 랭킹 등.
- 관계 태그: 로맨스, 혐관, 후회, 가족, 팀, 스승제자 등.
- 톤 태그: 어두움, 코미디, 피폐, 통쾌함, 잔잔함 등.

주의:
- 태그가 많을수록 좋은 것이 아니다.
- 서로 반대되는 기대를 만드는 태그는 조심한다.
- 주 장르 태그와 첫 5화 내용이 맞아야 한다.

## 무료분 전략

무료분의 목표는 모든 설정을 설명하는 것이 아니라 `믿게 만드는 것`이다.

초반 무료분에서 증명할 것:
- 주인공의 처지와 욕망.
- 핵심 소재의 실재성.
- 반복 보상 루프.
- 첫 적대 또는 첫 장애물.
- 작은 사이다나 의미 있는 생존.
- 더 큰 세계의 씨앗.

무료분에서 피할 것:
- 결제 전까지 보상을 미룬다.
- 초반 적대자를 끝없이 방치한다.
- 설정 설명 때문에 핵심 소재 사용이 늦다.
- 장기 떡밥만 열고 단기 답을 안 준다.

## 유료화 후보 지점

좋은 유료화 지점은 `보상을 준 직후`와 `더 큰 약속을 건 직후`가 겹치는 곳이다.

좋은 후보:
- 첫 주요 사이다 직후, 더 큰 적이 드러난다.
- 첫 큰 전투/시험/무대 승리 직후, 더 높은 판이 열린다.
- 주인공 비밀이 일부 드러난 직후, 대중/조직 반응이 시작된다.
- 관계가 전환된 직후, 더 큰 오해나 계약이 생긴다.
- 세계 규칙을 하나 이해한 직후, 그 규칙의 더 위험한 층이 열린다.

나쁜 후보:
- 아무 보상 없이 절단한다.
- 주인공이 계속 당하는 중에 결제를 요구한다.
- 유료화 직전까지 핵심 소재를 충분히 쓰지 않는다.
- 무료분에서 약속한 장르와 유료분 초반 장르가 달라진다.

## 장기 연재 확장

장기 연재는 무대를 키우되 초반 약속을 버리면 안 된다.

확장 방식:
- 개인 문제 -> 집단 문제.
- 생활권 적 -> 제도/조직 적.
- 작은 보상 -> 지위/영토/명성/책임.
- 단일 능력 -> 응용, 한계, 대가.
- 숨은 진실 -> 세계 규칙, 역사, 최종 가치관.

확장 점검:
- 초반 독자가 좋아한 보상이 후반에도 남아 있는가?
- 무대가 커져도 주인공 욕망이 선명한가?
- 새 인물과 세력이 기존 갈등을 대체하지 않고 확장하는가?
- 큰 사이다 후 다음 문제의 씨앗이 있는가?
