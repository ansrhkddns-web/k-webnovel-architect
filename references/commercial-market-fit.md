# 상업성 판단 모델

상업성은 작품을 평범하게 만드는 기준이 아니다. 독자가 클릭하기 전부터 결제 이후까지 `무엇을 기대하고, 무엇을 받으며, 왜 계속 따라오는지`를 선명하게 만드는 기준이다.

## 상업성 7요소

| 요소 | 질문 | 낮을 때 증상 | 보강 |
|---|---|---|---|
| 장르 명료도 | 제목과 1화에서 장르가 보이는가? | 독자가 어떤 재미인지 모른다 | 장르 키워드, 상황, 보상 단어를 앞에 둔다 |
| 클릭성 | 제목/소개글이 즉시 궁금한가? | 설명은 맞지만 손이 안 간다 | 모순, 처지, 능력, 위기 중 하나를 선명하게 한다 |
| 반복 보상 | 50화 이상 줄 보상이 있는가? | 초반 아이디어만 있고 금방 고갈된다 | 성장/관계/돈/지위/단서 사다리를 만든다 |
| 초반 유지력 | 1-5화 안에 재미가 증명되는가? | 설정만 있고 보상이 늦다 | 첫 사용, 첫 보상, 첫 적대를 앞당긴다 |
| 유료화 신뢰 | 무료분에서 이미 보상을 줬는가? | 결제 직전까지 기다리기만 한다 | 큰 보상 직후 더 큰 약속을 연다 |
| 장기 확장성 | 무대와 적이 커질 수 있는가? | 30화 뒤 할 일이 없다 | 개인 문제를 조직/사회/세계 문제로 확장한다 |
| 지식재산 확장성 | 웹툰/영상/오디오로 옮길 시각적 신호가 있는가? | 장면이 모두 내면 설명이다 | 상징 장면, 반복 공간, 시각적 능력, 관계 갈등을 만든다 |

## 상업성 점수표

각 항목을 1-5점으로 매긴다.

| 항목 | 1점 | 3점 | 5점 |
|---|---|---|---|
| 장르 명료도 | 장르가 안 보임 | 설명하면 이해됨 | 제목만 봐도 보임 |
| 클릭성 | 평범한 요약 | 한 가지 궁금증 | 즉시 누르고 싶은 모순/위기 |
| 반복 보상 | 10화도 불안 | 30-50화 가능 | 100화 이상 확장 가능 |
| 주인공 상품성 | 욕망이 흐림 | 응원 가능 | 처지, 욕망, 우위가 즉시 선명 |
| 보상 체감성 | 설명뿐 | 일부 장면화 | 매 보상이 몸/관계/지위로 보임 |
| 적대자 기능성 | 짜증 유발만 함 | 일부 기능 | 단계별 압박과 사이다 기대 |
| 초반 훅 | 늦음 | 1화 끝에 있음 | 첫 화면부터 압박과 이상성 |
| 유료화 지점 | 보상 전 절단 | 약간의 보상 뒤 절단 | 보상 직후 더 큰 약속 |
| 장기 떡밥 | 없음/방치 | 일부 단서 | 단기 답과 장기 질문 균형 |
| 플랫폼 적합도 | 독자층 불명 | 한 플랫폼 가능 | 여러 유통/확장 경로 가능 |

판정:
- 42점 이상: 상업 연재 기반이 강하다.
- 34-41점: 핵심은 있으나 패키징/초반 보강이 필요하다.
- 25-33점: 아이디어는 있으나 반복 보상과 유료화 신뢰가 약하다.
- 24점 이하: 장르 계약, 주인공 엔진, 회차 보상부터 다시 잡는다.

## 독자층별 핵심 보상

독자층은 성별이나 나이로 단정하지 않는다. `찾는 보상`으로 나눈다.

| 독자 욕망 | 좋아하는 보상 | 강한 장르 |
|---|---|---|
| 성장/상승 | 약자에서 강자, 랭크 상승, 인정 | 헌터, 무협, 스포츠, 아카데미, 직업물 |
| 사이다/복수 | 억울함 회수, 가해자 응징, 판 뒤집기 | 회귀, 복수, 학원, 재벌, 로판 악녀 |
| 전략/최적화 | 규칙 이해, 선점, 빌드, 경제적 판단 | 회귀, 게임, 탑, 사업, 대체역사 |
| 관계/감정 | 계약, 오해, 후회, 구원, 집착 | 로맨스, 로판, 비엘, 지엘, 가족물 |
| 생존/긴장 | 금기, 재난, 추격, 제한된 자원 | 아포칼립스, 공포, 스릴러, 미스터리 |
| 전문성/쾌감 | 실력 증명, 사건 해결, 업계 인정 | 의학, 법정, 요리, 연예계, 스포츠 |
| 운영/건설 | 거점, 영지, 회사, 국가, 팀 만들기 | 영지물, 경영물, 생존물, 대체역사 |
| 힐링/생활감 | 안정, 회복, 관계 복원, 일상 보상 | 힐링, 음식, 귀농, 가족, 일상 판타지 |

## 상업적 위험 신호

| 위험 | 설명 | 보완 |
|---|---|---|
| 문학적 제목 과잉 | 분위기는 좋지만 장르와 보상이 안 보인다 | 상업형 제목 후보를 별도로 만든다 |
| 설정 우선 전개 | 세계관이 장면보다 앞선다 | 설정은 현재 문제 해결에 필요한 만큼만 공개한다 |
| 주인공 무욕망 | 주인공이 그냥 휘말린다 | 외적 목표와 내적 결핍을 한 문장으로 고정한다 |
| 보상 불균형 | 고구마가 보상보다 길다 | 작은 사이다/성장/정보 보상을 끼운다 |
| 트로프 배신 | 독자가 산 장르 재미를 후반에 버린다 | 장르 전환 전 보상책과 예고를 깐다 |
| 차별화 과잉 | 익숙한 재미보다 낯섦이 먼저 온다 | 익숙한 장르 약속을 먼저 증명한다 |
| 초반 과밀 | 이름, 조직, 규칙이 너무 많다 | 1화에는 주인공, 압박, 이상성, 훅만 우선한다 |
| 유료화 절단감 | 보상 없이 결제벽이 온다 | 직전 보상과 다음 약속을 같이 배치한다 |

## 상업성 보강 순서

1. 제목과 한 줄 콘셉트에서 장르와 보상을 보이게 한다.
2. 1화 첫 화면에 압박 또는 이상성을 넣는다.
3. 5화 안에 첫 보상과 반복 루프를 증명한다.
4. 적대자 또는 장애물을 단계별로 만든다.
5. 무료분 끝에 중간 사이다와 더 큰 위기 씨앗을 둔다.
6. 장기 확장을 새 장르로 바꾸지 말고 기존 보상의 확대판으로 만든다.
