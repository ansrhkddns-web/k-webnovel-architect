# 콘셉트 검증 랩

콘셉트는 멋진 문장이 아니라 `연재 가능한 약속`이어야 한다. 이 문서는 아이디어가 상업 연재형 작품으로 버틸 수 있는지 검증한다.

## 7단계 검증

| 단계 | 질문 | 실패 신호 | 보정 |
|---|---|---|---|
| 장르 | 한 단어로 주 장르를 말할 수 있는가? | 장르 설명이 길다 | 주 장르를 하나 고른다 |
| 독자 | 누가 이걸 사러 들어오는가? | 모두가 대상이다 | 주 독자 페르소나를 정한다 |
| 처지 | 주인공의 시작 상태가 선명한가? | 평범한 사람이라고만 함 | 결핍, 굴욕, 욕망, 손실을 붙인다 |
| 우위 | 남들과 다른 무기가 있는가? | 그냥 열심히 함 | 특수 지식, 능력, 위치, 관계를 준다 |
| 루프 | 매화 반복할 사건이 있는가? | 첫 사건 뒤 할 일이 없다 | 반복 엔진을 만든다 |
| 보상 | 독자가 무엇을 기다리는가? | 분위기만 있다 | 성장/관계/돈/지위/단서 중 하나를 고정 |
| 확장 | 50화 뒤 무엇이 커지는가? | 초반 사건으로 끝난다 | 조직, 세계, 관계, 가치관을 확장한다 |

## 한 줄 콘셉트 압축

```text
[처지]의 주인공이 [특수 우위]로 [반복 사건]을 해결하며 [보상 사다리]를 오른다.
하지만 [차별화 twist/대가] 때문에 [장기 위험]이 커진다.
```

## 콘셉트 강도 테스트

| 테스트 | 질문 |
|---|---|
| 제목 연결 | 제목 후보가 콘셉트를 바로 떠올리게 하는가? |
| 1화 연결 | 1화 첫 장면으로 바꿀 수 있는가? |
| 5화 연결 | 5화 안에 반복 보상을 증명할 수 있는가? |
| 유료화 연결 | 첫 큰 보상 뒤 더 큰 약속이 열리는가? |
| 장기 연결 | 50화, 100화에서 무대가 커질 수 있는가? |
| IP 연결 | 그림/영상/오디오로 보이는 장면이 있는가? |

## 차별화 위치

차별화는 아래 중 하나에만 강하게 넣는 것이 좋다.

| 위치 | 예 |
|---|---|
| 주인공 처지 | 최약체인데 특정 조건에서만 강함 |
| 특수 우위 | 남들이 버린 능력을 다르게 해석 |
| 보상 조건 | 이기는 것보다 구조/기록/관계가 보상 조건 |
| 무대 | 흔한 장르 장치를 낯익은 생활권에 배치 |
| 적대자 | 적이 악해서가 아니라 제도적으로 정당함 |
| 장기 진실 | 초반 보상의 의미가 후반에 재해석됨 |

## 콘셉트 위험 등급

| 위험 | 설명 | 처방 |
|---|---|---|
| 문학적이지만 안 팔림 | 분위기는 있으나 장르/보상 불명 | 장르 키워드와 반복 보상 추가 |
| 클릭은 되나 유지 약함 | 제목은 강하지만 5화 루프 없음 | 초반 보상 루프 설계 |
| 초반만 강함 | 20화 이후 할 일이 없음 | 장기 사다리 추가 |
| 차별화 과잉 | 익숙한 재미가 보이지 않음 | 장르 약속 먼저 증명 |
| 너무 안전함 | 기존작 사이에서 묻힘 | twist 하나 추가 |
| 주인공 약함 | 욕망/선택이 흐림 | 비용 있는 반복 행동 부여 |
