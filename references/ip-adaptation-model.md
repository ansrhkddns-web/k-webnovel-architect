# 지식재산 확장 모델

상업 웹소설은 웹툰, 오디오, 드라마, 애니메이션, 게임, 굿즈 등으로 확장될 수 있다. 모든 작품이 확장을 목표로 할 필요는 없지만, 기획 단계에서 `장면성`, `관계성`, `상징성`을 확보하면 장기 가치가 커진다.

## 확장성 6요소

| 요소 | 질문 | 강화 방법 |
|---|---|---|
| 시각성 | 장면이 그림으로 바로 보이는가? | 반복 공간, 상징 아이템, 능력 연출을 만든다 |
| 캐릭터 실루엣 | 인물이 한눈에 구분되는가? | 역할, 말투, 욕망, 상징 행동을 분리한다 |
| 관계성 | 팬이 조합과 감정선을 말할 수 있는가? | 관계 전환점과 오해/선택을 단계화한다 |
| 장면 반복성 | 매체화 시 반복될 대표 장면이 있는가? | 시험, 전투, 무대, 사교전, 사건 해결 루프 |
| 세계 확장성 | 후속 시즌과 외전이 가능한가? | 조직, 지역, 과거, 다른 인물 축을 남긴다 |
| 팬덤 언어 | 독자가 짧게 부를 보상/상징이 있는가? | 별명, 기술명, 장소명, 관계명은 명확하게 만든다 |

## 매체별 적합 신호

| 매체 | 강한 신호 | 약한 신호 |
|---|---|---|
| 웹툰 | 시각적 능력, 강한 표정, 공간, 액션, 패션 | 내면 독백과 설정 설명 의존 |
| 드라마 | 관계 갈등, 직업/가족/사회 현실감, 배우가 살릴 캐릭터 | 세계관 비용이 너무 큼 |
| 애니메이션 | 액션, 판타지 능력, 몬스터, 과장된 연출 | 장면 변화가 적음 |
| 오디오 | 1인칭 몰입, 대사 힘, 공포/미스터리 분위기 | 시각 정보 없으면 이해 어려움 |
| 게임 | 성장 수치, 장비, 전투, 선택지, 수집 | 서사만 있고 시스템화 요소 부족 |
| 굿즈/팬덤 | 상징 아이템, 명대사, 캐릭터 조합 | 캐릭터 상징이 약함 |

## 장르별 IP 포인트

| 장르 | 확장 포인트 |
|---|---|
| 헌터/탑 | 전투 연출, 몬스터, 랭크, 장비 |
| 무협 | 초식, 문파 의상, 결투, 사제관계 |
| 로판 | 의상, 궁정, 관계 구도, 사교전 |
| BL/GL | 관계 조합, 대사, 감정 전환점 |
| 아이돌 | 무대, 팬덤, 음악/안무, 멤버 조합 |
| 스포츠 | 경기 장면, 라이벌, 팀 케미 |
| 전문직 | 사건 해결, 직업 공간, 명대사 |
| 공포/미스터리 | 금기, 상징 물건, 반전 구조 |
| 아포칼립스 | 거점, 생존 장비, 집단 갈등 |
| 영지/경영 | 지도, 건물, 조직 성장, 생산물 |

## 주의

- IP 확장을 위해 원작 초반 보상을 희생하지 않는다.
- 웹툰화 가능성을 높이겠다고 장면을 모두 설명형 콘티처럼 쓰지 않는다.
- 영상화를 의식해 장르 핵심 능력이나 세계관을 약하게 만들지 않는다.
- 팬덤용 상징은 억지로 만들기보다 반복 행동과 관계에서 자연스럽게 나오게 한다.
