# 트로프 조합 라이브러리

트로프는 작품을 낡게 만드는 장치가 아니라, 독자가 빠르게 재미를 이해하게 하는 신호다. 상업 기획에서는 트로프를 `출발 처지`, `특수 우위`, `반복 사건`, `보상 사다리`, `차별화 twist`로 나누어 조합한다.

## 조합 공식

```text
출발 처지
+ 특수 우위
+ 반복 사건 엔진
+ 보상 사다리
+ 차별화 twist
= 한 줄 콘셉트
```

## 출발 처지

| 처지 | 독자 감정 | 잘 맞는 장르 |
|---|---|---|
| 최약체 | 성장 기대 | 헌터, 무협, 아카데미, 스포츠 |
| 폐급/무능 취급 | 인정 욕구 | 시스템, 직업물, 아이돌 |
| 망나니/악명 | 평판 역전 | 로판, 판타지, 재벌, 무협 |
| 몰락 가문/파산 | 재건 욕구 | 로판, 경영, 영지, 대체역사 |
| 엑스트라/조연 | 생존과 원작 비틀기 | 빙의, 아카데미, 로판 |
| 회귀 전 실패자 | 복구와 복수 | 회귀, 재벌, 스포츠, 전문직 |
| 은퇴자/귀환자 | 숨은 강자 | 헌터, 무협, 군사, 직업물 |
| 빚/계약에 묶인 인물 | 탈출과 역전 | 로맨스, 사업, 범죄, 직업물 |
| 사회적 약자 | 사이다와 자립 | 복수, 로맨스, 현대판타지 |
| 천재지만 결핍 있음 | 능력 증명 | 전문직, 스포츠, 아이돌 |

## 특수 우위

| 우위 | 재미 | 위험 |
|---|---|---|
| 미래 지식 | 선점과 전략 | 전지성으로 긴장 붕괴 |
| 상태창/시스템 | 측정 가능한 성장 | 숫자 장부화 |
| 숨은 직업/스킬 | 차별화된 공략 | 설명 과밀 |
| 기연/비급/유산 | 한 방의 역전 | 편의주의 |
| 독점 공간/자원 | 비밀 성장 | 긴장 부족 |
| 전문 지식 | 현실감 있는 능력 | 강의화 |
| 원작 지식 | 플래그 회피 | 원작 설명 과잉 |
| 기억/기록 접근 | 미스터리와 전략 | 답을 너무 쉽게 얻음 |
| 특수 관계/계약 | 감정과 권력 긴장 | 계약이 장식화 |
| 사회적 권한 | 제도적 힘 | 주인공이 너무 편해짐 |
| 이상한 약점 | 차별화 | 보상보다 불편함이 큼 |

## 반복 사건 엔진

| 엔진 | 반복 구조 |
|---|---|
| 던전/게이트 | 진입 -> 위기 -> 공략 -> 전리품 -> 사회 반응 |
| 시험/결투 | 준비 -> 대결 -> 평가 -> 지위 변화 |
| 의뢰/사건 | 문제 접수 -> 전문 판단 -> 해결 -> 평판 상승 |
| 사교/정치 | 초대 -> 거래 -> 스캔들 -> 권력 변화 |
| 레이드/전쟁 | 정보 -> 배치 -> 전투 -> 보상/피해 |
| 방송/무대 | 준비 -> 공개 검증 -> 반응 -> 팬덤/안티 |
| 사업/투자 | 기회 발견 -> 자원 투입 -> 경쟁 -> 수익/위험 |
| 수사/추리 | 단서 -> 가설 -> 검증 -> 반전 |
| 생존/거점 | 자원 부족 -> 선택 -> 방어/확장 -> 내부 갈등 |
| 관계 계약 | 조건 -> 위반/오해 -> 선택 -> 감정 변화 |

## 보상 사다리

| 사다리 | 보상 예 |
|---|---|
| 힘 | 스탯, 경지, 스킬, 장비 |
| 지위 | 랭크, 직급, 작위, 학년/반, 문파 위치 |
| 돈 | 매출, 투자, 지분, 부동산, 보상금 |
| 관계 | 호감, 신뢰, 계약, 동맹, 가족 회복 |
| 정보 | 단서, 지도, 비밀, 원작 플래그 |
| 명성 | 조회수, 팬덤, 평판, 기사, 추천 |
| 영토/조직 | 가게, 길드, 영지, 회사, 팀, 거점 |
| 자유 | 빚 청산, 파혼, 탈출, 독립 |

## 차별화 twist

차별화는 하나만 강하게 넣는다.

| twist | 효과 | 주의 |
|---|---|---|
| 보상 조건이 특이함 | 익숙한 성장물에 새 규칙 | 너무 복잡하면 이탈 |
| 주인공 목표가 작음 | 생활감과 공감 | 장기 확장 설계 필요 |
| 적이 사회적으로 정당함 | 갈등 깊이 | 사이다가 늦어질 수 있음 |
| 능력이 대가를 요구함 | 긴장 유지 | 대가가 답답함만 주면 안 됨 |
| 무대가 낯익은 현실임 | 접근성 | 현실 반응 설계 필요 |
| 장르 보상을 다른 영역에 적용 | 신선함 | 주 장르 약속 유지 |
| 관계가 보상 조건임 | 감정과 사건 결합 | 관계 진전이 작위적이면 위험 |

## 장르 조합 패턴

| 조합 | 주 보상 | 주의 |
|---|---|---|
| 헌터 + 회귀 | 공략 선점과 랭크 상승 | 미래 지식으로 전투 긴장 붕괴 금지 |
| 헌터 + 학원 | 경쟁 공간과 공개 평가 | 학교 규칙이 실제로 작동해야 함 |
| 던전 + 경영 | 자원 독점과 시장 확장 | 경영이 숫자만 되지 않게 장면화 |
| 무협 + 현판 | 무공 위계와 현대 반응 | 두 세계 규칙의 충돌을 명확히 |
| 무협 + 의학 | 무공/의술로 생명 구함 | 설명 강의화 주의 |
| 로판 + 복수 | 지위 역전과 감정 대가 | 복수 후 관계 목표 필요 |
| 로판 + 경영 | 사회적 독립과 돈 | 사업이 장식화되지 않게 |
| 악녀 + 계약결혼 | 평판 역전과 관계 긴장 | 계약 조건이 사건을 만들어야 함 |
| 아이돌 + 회귀 | 무대 선점과 팬덤 성장 | 미래 지식보다 실력 증명 필요 |
| 스포츠 + 시스템 | 기록 성장과 빌드 | 시스템이 경기 선택을 대신하지 않게 |
| 전문직 + 미스터리 | 사건 해결과 실력 인정 | 전문 설명보다 판단 장면 우선 |
| 아포칼립스 + 영지 | 생존과 공동체 건설 | 운영 설명 과밀 주의 |
| 괴담 + 루프 | 규칙 발견과 생존 | 반복마다 새 정보 필요 |
| 비엘/지엘 + 오피스 | 관계 긴장과 사회적 위험 | 업무 사건이 관계를 움직여야 함 |
