# 회차 로드맵

회차표는 줄거리 요약표가 아니다. 각 화가 독자에게 어떤 보상을 주고, 어떤 변화를 만들고, 왜 다음 화를 누르게 하는지 정하는 설계도다.

## 회차 설계 원칙

모든 회차에는 아래 네 가지가 있어야 한다.

- 상황 변화: 끝났을 때 주인공의 처지가 달라졌는가?
- 독자 보상: 성장, 승리, 정보, 관계, 지위, 감정 해방 중 무엇을 받는가?
- 다음 질문: 다음 화를 눌러야 하는 구체적 이유가 있는가?
- 장기 기능: 이 회차가 시즌, 떡밥, 인물 변화에 어떤 역할을 하는가?

설명만 하는 회차, 이동만 하는 회차, 기다리기만 하는 회차, 같은 보상을 반복하는 회차를 피한다.

## 회차 카드

```markdown
### N화. [가제]
- 회차 기능:
- 시작 압박:
- 주인공 목표:
- 장애물:
- 선택/행동:
- 보상/변화:
- 체감 증거:
- 주변 반응:
- 심거나 회수한 떡밥:
- 이탈 위험:
- 엔딩 훅:
```

## 1화

목표: 제목과 1화만으로 장르 약속을 알게 한다.

포함한다:
- 주인공의 현재 압박 또는 결핍
- 비정상 사건, 특수한 우위, 되돌릴 수 없는 변화 중 하나
- 장르 보상의 예고
- 다음 화를 누르게 하는 구체적 질문

피한다:
- 긴 세계사 프롤로그
- 능력 체계 강의
- 이름 많은 조연 소개
- 주인공이 관찰만 하고 선택하지 않는 전개
- 장르 핵심 소재가 너무 늦게 나오는 전개

## 1화 구성 카드

```markdown
## 1화 구성
- 첫 화면 압박:
- 주인공 결핍:
- 장르 핵심 소재:
- 되돌릴 수 없는 변화:
- 독자가 받을 첫 보상:
- 마지막 훅:
- 2화에서 바로 확인할 것:
```

1화 마지막 훅은 추상적인 분위기보다 구체적인 행동, 위협, 숫자, 선택, 증거가 좋다.

## 2-5화

목표: 이 작품의 반복 재미를 증명한다.

포함한다:
- 주인공이 핵심 우위를 한 번 이상 사용한다.
- 첫 작은 승리, 생존, 관계 변화, 지위 변화 중 하나가 나온다.
- 반복 문제의 원천이 보인다.
- 조력자, 라이벌, 적대자 중 최소 하나의 기능이 보인다.
- 장기 떡밥의 첫 단서가 심긴다.

독자 질문:
`이 작품이 주는 재미를 알겠다. 이걸 더 보고 싶은가?`

## 2-5화 기능 배치

초반 5화 안에는 아래 기능을 가능한 한 배치한다. 순서는 장르에 따라 달라질 수 있다.

| 기능 | 역할 |
|---|---|
| 첫 재시도 | 주인공이 핵심 소재에 다시 손을 댄다 |
| 첫 규칙 확인 | 세계/능력/관계의 작동 조건을 본다 |
| 첫 작은 보상 | 독자가 계속 읽을 근거를 얻는다 |
| 첫 외부 압박 | 적, 제도, 가족, 관계가 주인공을 밀어붙인다 |
| 첫 장기 단서 | 이야기가 더 커질 여지를 심는다 |

## 6-10화

목표: 첫 보상의 비용과 첫 진짜 장애물을 보여준다.

포함한다:
- 초반 승리 때문에 생긴 후폭풍
- 더 강한 적, 조직, 제도, 가족, 라이벌, 던전, 사건 중 하나
- 주인공 방식의 대가
- 1부 목표의 구체화
- 보상 구조를 독자가 이해할 수 있는 실험 또는 체감 장면

## 6-10화 기능 배치

| 기능 | 역할 |
|---|---|
| 보상 검증 | 보상이 실제 문제를 해결하는지 확인 |
| 대가 노출 | 성장 방식이 공짜가 아님을 보여줌 |
| 적대 강화 | 상대가 주인공 변화를 눈치챔 |
| 목표 구체화 | 1부에서 무엇을 해결할지 선명해짐 |
| 변주 시작 | 같은 보상을 다른 상황에 적용 |

## 11-25화

목표: 첫 주요 아크를 완성하거나 유료 전환 직전 신뢰를 만든다.

포함한다:
- 주인공의 첫 의미 있는 승리
- 공개 인정, 관계 변화, 복수, 랭크 상승, 돈, 명성 중 하나
- 더 큰 세계나 더 깊은 적을 여는 정보
- 유료화 후보 지점

주의:
- 유료 전환 시점이 “이제야 이야기가 시작된다”처럼 느껴지면 늦다.
- 유료 전환 전에는 독자가 이미 여러 번 보상을 받아 신뢰한 상태여야 한다.

## 초반부 기능 리듬

초반부는 보통 아래 리듬 중 하나를 따른다.

### 성장형

```text
압박 -> 첫 생존/각성 -> 규칙 실험 -> 작은 역전 -> 외부 의심 -> 첫 아크 사이다
```

### 복수형

```text
불의 -> 무력함 -> 반격 수단 확보 -> 증거/명분 축적 -> 작은 응징 -> 큰 응징 예고
```

### 로맨스/관계형

```text
사회적 위기 -> 위험한 관계 시작 -> 거래/오해 -> 감정 균열 -> 공개 선택 -> 더 큰 스캔들
```

### 미스터리형

```text
이상 현상 -> 첫 조사 -> 작은 답 -> 더 위험한 질문 -> 금기 위반 -> 생존/진실 일부 회수
```

## 26-50화

목표: 소재가 오래 갈 수 있음을 증명한다.

포함한다:
- 새 무대 또는 더 넓은 세력도
- 적대자의 두 번째 층
- 자원 경쟁, 정치, 여론, 가족, 조직 같은 사회적 비용
- 초반 떡밥 최소 하나의 부분 회수
- 더 큰 사이다 뒤의 다음 위기 씨앗

## 51-100화

목표: 신화, 정체성, 관계, 세계의 장기 목표를 깊게 만든다.

포함한다:
- 주인공 우위나 세계 규칙의 숨은 진실
- 관계 역전 또는 신뢰 붕괴
- 적 규모 확장
- 새 전략적 한계
- 장기 목표 재정의

## 아크 카드

```markdown
## [아크명]
- 회차 범위:
- 아크 질문:
- 독자 보상:
- 주 압박:
- 주인공 목표:
- 적대자/세력:
- 중간 반전:
- 클라이맥스 선택:
- 보상:
- 다음 질문:
- 이탈 위험:
```

## 시즌 카드

```markdown
| 시즌 | 권장 화수 | 핵심 약속 | 큰 보상 | 다음 위기 씨앗 |
|---|---|---|---|---|
| 1 |  |  |  |  |
| 2 |  |  |  |  |
```

## 보상 간격

- 1화: 클릭 이유와 비정상성.
- 2-5화: 반복 재미 증명.
- 2-4화마다: 작은 보상.
- 8-12화마다: 중간 사이다 또는 큰 관계 변화.
- 25-40화마다: 국면 전환.
- 큰 사이다 직후: 다음 위기 씨앗.

## 유료화 지점 점검

```markdown
| 후보 | 직전 보상 | 새로 열린 위기 | 독자가 믿을 근거 | 위험 |
|---|---|---|---|---|
| A |  |  |  |  |
| B |  |  |  |  |
```

유료화 직전에는 보상을 끊기만 하지 않는다. 독자가 이미 받은 보상 때문에 다음 보상도 믿게 만들어야 한다.

## 회차별 보상 유형

| 보상 | 독자가 느끼는 재미 |
|---|---|
| 성장 | 이전보다 강해지거나 더 잘한다 |
| 승리 | 적, 제도, 상황을 이긴다 |
| 정보 | 몰랐던 규칙이나 진실을 안다 |
| 관계 | 신뢰, 의심, 집착, 애정이 움직인다 |
| 지위 | 평판, 랭크, 소속, 권한이 바뀐다 |
| 자원 | 돈, 아이템, 땅, 조직, 도구가 생긴다 |
| 감정 해방 | 억울함, 수치심, 분노가 풀린다 |
| 위기 | 더 보고 싶게 만드는 새 위험이 열린다 |

## 템포 진단

- 초반이 느리면 핵심 소재, 위기, 특수성을 1화 안쪽으로 당긴다.
- 중반이 처지면 규칙이 바뀌는 중간 반전을 넣는다.
- 주인공이 쉽게 이기면 승리의 사회적 후폭풍을 만든다.
- 세계가 좁아지면 기존 약속을 유지한 채 더 큰 무대를 연다.
- 장기 미스터리를 잊을 만하면 5-10화마다 작게 상기시킨다.
- 고구마가 길어지면 작은 사이다, 정보 보상, 성장 체감 중 하나를 넣는다.

## 회차표 완성 검문

- 각 화 제목만 봐도 변화 방향이 보이는가?
- 보상/변화 칸이 비어 있는 화가 없는가?
- 엔딩 훅이 “불길했다”처럼 추상적이지 않은가?
- 같은 적대 방식이 3화 이상 반복되지 않는가?
- 설정 설명 화에도 관계, 자원, 위험 중 하나의 변화가 있는가?
- 큰 보상 직후 다음 위험 씨앗이 있는가?
