# 원고 제작 브리프

원고 제작 브리프는 회차 브리프보다 한 단계 더 구체적인 집필 지시서다. 실제 원고를 쓰거나 다른 쓰기 스킬에 넘길 때 사용한다.

## 제작 브리프 구조

```markdown
## N화 원고 제작 브리프

### 목표 독자 반응
-

### 회차 기능
-

### 장면 구성
| 순서 | 장면 | 기능 | 감정 | 정보/보상 |
|---|---|---|---|---|
| 1 |  |  |  |  |
| 2 |  |  |  |  |
| 3 |  |  |  |  |

### 반드시 들어갈 요소
-

### 절대 피할 요소
-

### 대사/관계 포인트
-

### 액션/판단 포인트
-

### 설정 공개 범위
-

### 엔딩 문장 방향
-

### 다음 화 인계
-
```

## 장면 구성 원칙

- 한 장면에는 하나의 작은 승부가 있어야 한다.
- 장면마다 압박, 선택, 결과 중 하나가 있어야 한다.
- 설명 장면도 관계 변화나 정보 보상이 있어야 한다.
- 대사는 정보 전달보다 권력 관계와 감정 변화를 보여줘야 한다.
- 액션 장면은 동작, 결과, 대가를 분명히 한다.

## 정보 공개 범위

```markdown
## 설정 공개 제한
- 이번 화에서 독자가 알아도 되는 것:
- 주인공만 아는 것:
- 독자는 눈치챌 수 있지만 확정하지 않을 것:
- 아직 공개하지 않을 것:
```

## 엔딩 훅 방향

| 훅 | 사용 상황 |
|---|---|
| 보상 직후 대가 | 성장물, 시스템, 헌터 |
| 관계 파열 | 로맨스, 로판, BL/GL |
| 증거 등장 | 복수, 수사, 미스터리 |
| 상위 적 개입 | 무협, 헌터, 기업물 |
| 금기 위반 | 공포, 오컬트, 데스게임 |
| 공개 반응 | 아이돌, 스포츠, 전문직 |

## 원고 인계 체크

- 회차 기능이 명확한가?
- 장면 수가 과도하지 않은가?
- 첫 장면이 이전 화 훅에 반응하는가?
- 설정 공개 범위가 제한되어 있는가?
- 엔딩 훅이 다음 화의 첫 장면으로 이어지는가?
