# 장르 레퍼런스 매트릭스

이 문서는 특정 작품명을 저장하지 않는다. 여러 한국 웹소설 장르물에 적용할 수 있는 `장르 슬롯`, `독자 보상`, `필수 엔진`, `이탈 위험`을 정리한다.

## 사용법

1. 주 장르를 하나 고른다.
2. 보조 장르를 0-2개만 고른다.
3. 주 장르가 독자 약속을 소유하게 한다.
4. 보조 장르는 주 장르의 보상을 강화하는 방향으로만 쓴다.
5. 회차표에는 주 장르 보상을 먼저 배치한다.
