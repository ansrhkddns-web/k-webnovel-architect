# 인물 기능 모델

웹소설 인물은 매력도 중요하지만, 장기 연재에서는 `회차에서 어떤 일을 하느냐`가 더 중요하다. 인물을 만들 때 성격표보다 서사 기능을 먼저 정한다.

## 주인공 엔진

```markdown
## 주인공
- 현재 처지:
- 외적 욕망:
- 내적 결핍:
- 특수한 우위:
- 우위의 대가:
- 반복 행동:
- 도덕적 선:
- 독자가 응원할 이유:
- 장기 변화:
```

좋은 주인공은 매화 아래 중 하나를 한다.

- 위험을 감수한다.
- 새 규칙을 실험한다.
- 적의 판을 읽고 바꾼다.
- 관계에서 더 이상 예전처럼 굴지 않는다.
- 얻은 보상을 다음 문제 해결에 쓴다.

## 적대자 사다리

적대자는 주인공을 괴롭히는 사람이 아니라, 주인공의 성장을 검증하는 장치다.

| 단계 | 기능 | 예시 기능 |
|---|---|---|
| 생활권 적 | 초반 압박과 감정 명분 | 조롱, 폭력, 빚, 차별, 가족 압박 |
| 기술/능력 적 | 성장 루프 검증 | 더 강한 전투, 더 어려운 의뢰, 더 높은 시험 |
| 사회적 적 | 지위와 평판 압박 | 조직, 학교, 가문, 회사, 길드, 언론 |
| 가치관 적 | 주인공 선택의 의미를 공격 | 생존 논리, 질서 논리, 혈통 논리, 효율 논리 |
| 장기 적 | 세계관의 큰 문제와 연결 | 재난, 흑막, 제도, 신화, 최종 세력 |

## 적대자 기능 카드

```markdown
## [적대자 또는 적대 기능]
- 압박 방식:
- 겉명분:
- 실제 욕망:
- 주인공의 어떤 약점을 찌르는가:
- 주인공의 어떤 성장을 검증하는가:
- 독자 사이다 기대:
- 퇴장/전환 조건:
- 퇴장 후 남길 씨앗:
```

## 조력자 기능

조력자는 주인공을 편하게 만들기만 하면 안 된다. 조력자는 정보, 대가, 감정, 갈등 중 하나를 가져와야 한다.

| 유형 | 기능 | 위험 | 보정 |
|---|---|---|---|
| 정보 조력자 | 규칙, 단서, 시장, 조직 정보를 준다 | 설명 담당으로 굳는다 | 자기 이해관계와 오판을 준다 |
| 감정 조력자 | 주인공의 인간성을 붙잡는다 | 이야기에서 기능이 약하다 | 주인공 선택에 실제 영향을 주게 한다 |
| 기술 조력자 | 훈련, 제작, 치료, 해킹, 행정 등 실무 | 편의 장치가 된다 | 조건, 비용, 실패 가능성을 둔다 |
| 대비 조력자 | 주인공과 다른 선택을 보여준다 | 단순 해설자가 된다 | 독자적으로 욕망과 위기를 준다 |
| 배신 가능 조력자 | 긴장과 의심을 만든다 | 억지 반전이 된다 | 초반부터 이해관계 단서를 심는다 |

## 라이벌 기능

라이벌은 주인공을 닮았지만 다른 선택을 하는 인물이다.

```markdown
## 라이벌
- 주인공과 닮은 점:
- 주인공과 다른 선택:
- 더 앞선 영역:
- 밀리는 영역:
- 경쟁 보상:
- 협력 가능성:
- 타락 또는 성장 가능성:
```

라이벌이 필요한 경우:
- 주인공이 너무 쉽게 강해질 때.
- 성장 기준을 외부에서 보여줘야 할 때.
- 같은 목표를 다른 방식으로 추구하는 인물이 필요할 때.
- 장기적으로 동료, 적, 거울상 중 하나가 필요할 때.

## 감정/로맨스 축

관계는 분위기가 아니라 사건을 움직여야 한다.

```markdown
## 감정축
- 첫 인상:
- 끌림 또는 관심의 이유:
- 저항의 이유:
- 관계가 주인공 목표에 주는 도움:
- 관계가 만드는 위험:
- 전환점:
- 파국 또는 결실 조건:
```

관계 변화의 기본 단계:

1. 인식: 서로를 다르게 보기 시작한다.
2. 거래: 도움, 정보, 보호, 이용 중 하나가 생긴다.
3. 오해: 겉행동과 속욕망이 어긋난다.
4. 선택: 관계 때문에 손해를 감수하거나 버린다.
5. 재정의: 조력자, 연인, 적, 동맹, 경쟁자로 위치가 바뀐다.

## 외부 관찰자

성장물에서 외부 관찰자는 주인공 변화를 독자에게 확인시킨다.

| 관찰자 | 확인하는 변화 |
|---|---|
| 가족/생활권 인물 | 건강, 습관, 분위기, 비밀 |
| 동급자/동료 | 실력, 태도, 평판 |
| 적대자 | 위협 수준, 질투, 의심 |
| 전문가/기관 | 규칙 위반, 능력의 비정상성 |
| 대중/여론 | 명성, 오해, 추종, 배척 |

외부 관찰자는 너무 빨리 진실을 알아내면 비밀 긴장이 무너진다. 처음에는 `이상함`, 다음에는 `의심`, 그다음에 `증거`로 단계화한다.

## 인물 과밀 방지

인물이 많아질 때는 아래 기준으로 합치거나 줄인다.

- 같은 방식으로 압박하는 적은 합친다.
- 같은 정보를 주는 조력자는 합친다.
- 이름은 있는데 회차 기능이 없으면 삭제하거나 배경 인물로 낮춘다.
- 장기 기능이 없는 인물은 첫 아크 안에서 역할을 끝낸다.
- 로맨스/감정축은 주 장르가 아니면 너무 많은 후보를 만들지 않는다.

```markdown
| 인물 | 기능 | 주는 압박/보상 | 첫 사용 | 전환점 | 퇴장/장기 역할 |
|---|---|---|---|---|---|
|  |  |  |  |  |  |
```

점검 질문:
- 주인공을 움직이게 만드는 인물이 있는가?
- 주인공 변화를 목격하는 인물이 있는가?
- 주인공의 약점을 찌르는 인물이 있는가?
- 독자가 기다릴 사이다 대상이 있는가?
- 장기적으로 다시 돌아올 씨앗이 있는가?
