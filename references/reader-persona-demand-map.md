# 독자 페르소나와 수요 지도

상업 웹소설에서 독자층은 나이/성별보다 `어떤 보상을 사러 들어오는가`로 보는 편이 더 실용적이다. 이 문서는 작품의 핵심 독자층, 클릭 동기, 결제 동기, 이탈 원인을 분류한다.

## 독자 보상 페르소나

| 페르소나 | 구매 욕망 | 클릭 신호 | 계속 읽는 이유 | 이탈 원인 |
|---|---|---|---|---|
| 성장 추적형 | 약자가 강해지는 과정을 본다 | 최약체, 폐급, 성장, 스탯, 랭크 | 변화가 수치와 장면으로 보임 | 성장 무체감, 반복 전투 |
| 사이다 추구형 | 억울함이 뒤집히는 걸 본다 | 복수, 참교육, 역전, 후회 | 빌런이 대가를 치름 | 고구마 과잉, 응징 지연 |
| 최적화 공략형 | 규칙을 이해하고 선점하는 맛 | 회귀, 공략, 히든, 빌드, 선점 | 주인공이 남보다 똑똑하게 움직임 | 정보 만능, 긴장 붕괴 |
| 지위 상승형 | 낮은 평판이 뒤집히는 맛 | 망나니, 악녀, 폐급, 재벌, 수석 | 주변 태도와 권한이 바뀜 | 평판 변화가 장면화되지 않음 |
| 관계 몰입형 | 관계의 긴장과 선택을 본다 | 계약, 후회, 혐관, 구원, 집착 | 감정 변화가 사건에 영향 | 관계가 분위기만 있음 |
| 전문성 감탄형 | 실력자가 현장에서 증명하는 맛 | 의사, 변호사, 셰프, 천재, 사건 | 문제 해결 장면이 구체적 | 설명 강의화 |
| 생존 긴장형 | 제한된 상황에서 버티는 맛 | 아포칼립스, 생존, 괴담, 재난 | 선택마다 손실과 보상이 있음 | 고통만 누적됨 |
| 운영 건설형 | 조직/영지/가게가 커지는 맛 | 영지, 경영, 거점, 제작, 농사 | 생산, 방어, 사람 관리가 확장 | 행정 설명 과밀 |
| 무대 응원형 | 무대/경기에서 인정받는 맛 | 아이돌, 배우, 스포츠, 방송 | 공개 평가와 팬/팀 반응 | 반응이 작위적 |
| 미스터리 추적형 | 단서와 반전을 따라가는 맛 | 괴담, 추리, 비밀, 규칙, 루프 | 작은 답과 큰 질문이 교대 | 답을 너무 안 줌 |
| 힐링 안정형 | 회복과 안전감, 생활 보상 | 힐링, 귀농, 가족, 음식, 일상 | 작은 변화와 정서적 안정 | 갈등이 없거나 목표가 흐림 |

## 독자 계약 문장

```markdown
이 작품의 핵심 독자는 [페르소나]다.
이 독자는 [클릭 신호]를 보고 들어오며,
초반에는 [1-5화 보상]을 기대하고,
무료분 끝에서는 [중간 보상/큰 약속]을 확인해야 결제한다.
이 독자가 가장 싫어할 이탈 요소는 [이탈 원인]이다.
```

## 페르소나 조합

주 페르소나는 하나만 정하고, 보조 페르소나는 최대 두 개까지 둔다.

좋은 조합:
- 성장 추적형 + 사이다 추구형.
- 관계 몰입형 + 지위 상승형.
- 전문성 감탄형 + 미스터리 추적형.
- 생존 긴장형 + 운영 건설형.
- 최적화 공략형 + 성장 추적형.

위험한 조합:
- 힐링 안정형 + 피폐 생존형: 톤 약속이 충돌할 수 있다.
- 관계 몰입형 + 무감정 먼치킨형: 감정 보상과 압도감이 서로 약해질 수 있다.
- 미스터리 추적형 + 빠른 사이다형: 답을 숨기는 재미와 즉시 보상이 충돌할 수 있다.

## 독자 반응 예측

| 기획 요소 | 긍정 반응 | 부정 반응 |
|---|---|---|
| 초반 압박 | 주인공을 응원하게 됨 | 너무 길면 답답함 |
| 첫 보상 | 계속 읽을 근거 | 작동 방식이 안 보이면 약함 |
| 빌런 | 사이다 기대 | 빌런 운빨 생존은 이탈 |
| 설정 | 세계가 커 보임 | 초반 과밀은 피로 |
| 로맨스/관계 | 감정 몰입 | 사건과 분리되면 장식 |
| 유료화 훅 | 다음 보상 기대 | 보상 전 절단은 불신 |

## 출력 형식

```markdown
## 독자 페르소나

### 주 페르소나
-

### 보조 페르소나
-

-

-

### 결제 전 확인시켜야 할 신뢰
-

### 가장 위험한 이탈 요소
-
```
