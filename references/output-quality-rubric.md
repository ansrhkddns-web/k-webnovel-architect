# 출력 품질 루브릭

이 문서는 `evaluation-prompts.md`의 테스트 결과를 채점하기 위한 기준이다. 좋은 답변은 예쁘고 길기만 한 답변이 아니라, 사용자가 바로 기획과 수정에 쓸 수 있는 답변이다.

## 채점 방식

각 항목을 0-5점으로 평가한다.

| 점수 | 의미 |
|---|---|
| 0 | 항목이 없다 |
| 1 | 이름만 언급한다 |
| 2 | 일반론은 있으나 작품에 적용되지 않는다 |
| 3 | 적용은 되었으나 처방이 약하다 |
| 4 | 실전적으로 쓸 수 있다 |
| 5 | 구체적이고 상업적 판단까지 포함한다 |

## 핵심 루브릭

| 항목 | 질문 |
|---|---|
| 요청 적합성 | 사용자가 요청한 산출물을 빠뜨리지 않았는가? |
| 장르 명료도 | 주 장르와 보조 장르를 분리했는가? |
| 독자 페르소나 | 누가 왜 읽는지 구매 욕망으로 설명했는가? |
| 상업성 판단 | 클릭성, 반복 보상, 무료분, 유료화 신뢰를 봤는가? |
| 콘셉트 검증 | 1화, 5화, 유료화, 50화 확장으로 이어지는가? |
| 트로프 조합 | 처지, 우위, 반복 사건, 보상 사다리, 차별화를 나눴는가? |
| 회차 기능 | 회차별 보상/변화/엔딩 훅이 있는가? |
| 체감성 | 성장/관계/지위 변화가 장면으로 확인되는가? |
| 이탈 위험 | 후킹 지연, 보상 지연, 설정 과밀, 반복 피로 등을 찾았는가? |
| 수정 처방 | 문제를 실제 수정 지시로 바꿨는가? |
| 고유 데이터 보호 | 레퍼런스의 고유 설정/문장/사건을 복사하지 않았는가? |

## 산출물별 필수 조건

### 빠른 기획서

- 장르 슬롯.
- 독자 페르소나.
- 한 줄 콘셉트 3개 이상.
- 제목 후보.
- 주인공 욕망과 특수 우위.
- 반복 보상 루프.
- 첫 5화 목표.
- 가장 큰 이탈 위험.

### 전체 바이블

- 상업성 진단.
- 장르 계약.
- 콘셉트 검증.
- 인물 기능.
- 성장 루프.
- 회차 로드맵.
- 유료화 후보.
- 장기 떡밥.
- 연속성 관리.
- 품질 게이트.

### 원고/기획 리뷰

- 치명 문제를 먼저 제시.
- 점수표.
- 구간별 이탈 위험.
- 회차별 기능 분석.
- 수정 우선순위.
- 다음 작업 지시.

### 제목/소개글 생성

- 제목 유형을 나눔.
- 제목별 장점/위험.
- 20자/60자/150자/400자 중 요청한 길이 반영.
- 태그 묶음.
- 소개글 과장 여부 점검.

### 회차 집필 지시서

- 회차 기능.
- 시작 압박.
- 필수 장면.
- 금지할 설명.
- 독자 보상.
- 체감 장면.
- 엔딩 훅.
- 다음 화 연결.

## 실패 패턴

| 실패 | 설명 |
|---|---|
| 장르 일반론 | 장르 설명만 있고 해당 기획에 적용하지 않음 |
| 제목만 많음 | 제목 후보는 많지만 왜 좋은지 판단이 없음 |
| 세계관 과잉 | 설정 확장만 하고 1화/5화 보상이 없음 |
| 독자 부재 | 누가 왜 읽는지 없음 |
| 유료화 부재 | 무료분 이후 결제 설계가 없음 |
| 수정 불가 | 문제를 지적하지만 어떻게 고칠지 없음 |
| 과도한 확신 | 최신 시장 데이터가 없는데 현재 랭킹처럼 말함 |
