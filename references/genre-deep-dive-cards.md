# 장르 심화 카드

`genre-reference-matrix.md`가 넓은 장르 지도를 제공한다면, 이 문서는 실제 기획에 바로 쓰기 위한 장르별 심화 카드다. 각 장르는 `독자 구매 이유`, `초반부 증명`, `반복 보상`, `상업 리스크`, `유료화 훅`을 따로 본다.

## 카드 형식

```markdown
## [장르]
- 구매 이유:
- 제목 신호:
- 1화에서 보여줄 것:
- 5화 안에 증명할 것:
- 반복 보상:
- 인물 기능:
- 유료화 훅:
- 장기 확장:
- 이탈 위험:
- 보정:
```

## 헌터 / 게이트

- 구매 이유: 현대 사회에서 측정 가능한 힘과 지위 상승을 본다.
- 제목 신호: 등급, 스킬, 헌터, 게이트, 던전, 각성, 랭크, 공략.
- 1화에서 보여줄 것: 주인공의 낮은 처지, 게이트/던전 위기, 특수한 우위의 첫 단서.
- 5화 안에 증명할 것: 첫 공략, 첫 보상, 기존 평가와 다른 결과.
- 반복 보상: 능력 상승, 아이템, 랭크, 길드/국가/언론 반응.
- 인물 기능: 라이벌 헌터, 길드 문지기, 기관 감시자, 구조받는 일반인, 숨은 강자.
- 유료화 훅: 첫 큰 공략 성공 직후 더 큰 게이트나 비밀 등급 노출.
- 장기 확장: 개인 던전 -> 길드/국가 -> 세계급 재난.
- 이탈 위험: 전투 반복, 상태창 과잉, 사회 반응 부재.
- 보정: 전투 외에 구조, 정치, 자원 경쟁, 여론, 가족 반응을 섞는다.

## 탑 / 시련

- 구매 이유: 단계 돌파, 규칙 공략, 랭킹 상승.
- 제목 신호: 탑, 층, 시련, 공략, 히든, 랭커, 보상.
- 1화에서 보여줄 것: 탑 진입 이유, 첫 규칙, 실패 시 대가.
- 5화 안에 증명할 것: 남들과 다른 공략법과 첫 보상.
- 반복 보상: 층 돌파, 히든 조건, 랭킹, 아이템, 동료 역할.
- 인물 기능: 경쟁자, 배신자, 정보상, 운영자/관리자, 층 보스.
- 유료화 훅: 예상 규칙을 깨는 새 층 또는 랭킹 공개.
- 장기 확장: 개인 생존 -> 파티/길드 -> 탑의 목적.
- 이탈 위험: 층 구조 반복, 퀘스트 임의성.
- 보정: 층마다 시험하는 약점과 보상 종류를 바꾼다.

## 던전 독점 / 비밀 자원

- 구매 이유: 남들이 모르는 자원을 혼자 선점하는 쾌감.
- 제목 신호: 나만, 독점, 비밀, 숨겨진, 집/가게/창고/문, 자원.
- 1화에서 보여줄 것: 비밀 자원의 발견과 위험.
- 5화 안에 증명할 것: 실제 보상과 숨겨야 할 이유.
- 반복 보상: 탐사, 전리품, 성장, 시장 가치, 비밀 유지.
- 인물 기능: 의심하는 주변인, 자원 노리는 적, 거래 상대, 외부 관찰자.
- 유료화 훅: 비밀이 일부 노출되거나 자원의 더 큰 정체가 드러남.
- 장기 확장: 개인 독점 -> 시장/조직/세계적 가치.
- 이탈 위험: 너무 편리한 보상 상자.
- 보정: 위험, 비용, 추적, 오염, 관리 문제를 붙인다.

## 회귀 / 재벌 / 사업

- 구매 이유: 미래 지식으로 돈과 권력을 선점하고 억울함을 갚는다.
- 제목 신호: 회귀, 막내, 재벌, 투자, 주식, 회사, 계약, 인수.
- 1화에서 보여줄 것: 실패한 과거와 되돌릴 기회.
- 5화 안에 증명할 것: 첫 선점, 첫 변수, 첫 돈/평판 변화.
- 반복 보상: 투자 수익, 지분, 계약, 조직 장악, 복수 단계.
- 인물 기능: 가족 경쟁자, 기업 문지기, 언론, 법률/회계 조력자, 배신자.
- 유료화 훅: 첫 큰 계약 성공 직후 더 큰 경쟁자 등장.
- 장기 확장: 개인 복수 -> 그룹 경영 -> 산업/정치 충돌.
- 이탈 위험: 숫자 설명 과잉, 미래 지식 만능.
- 보정: 협상 장면, 공개 결과, 예상 밖 변수, 법적/평판 비용을 둔다.

## 무협 / 무림

- 구매 이유: 무공 위계, 은원, 체면, 복수, 사문 상승.
- 제목 신호: 천마, 검, 무공, 문파, 사부, 회귀, 비급, 절대자.
- 1화에서 보여줄 것: 굴욕, 은원, 무공 정체성.
- 5화 안에 증명할 것: 첫 무공 사용 또는 첫 결투의 변화.
- 반복 보상: 경지, 초식, 명성, 문파 지위, 원한 회수.
- 인물 기능: 사형제, 사부, 문파 문지기, 정적, 은원 상대.
- 유료화 훅: 첫 공개 결투 승리 직후 더 큰 문파/세력 노출.
- 장기 확장: 개인 복수 -> 문파 재건 -> 무림 질서 재편.
- 이탈 위험: 문파명 과밀, 수련 요약.
- 보정: 세력은 설명보다 충돌로 소개하고 수련은 장면의 문제로 만든다.

## 선협 / 수선

- 구매 이유: 경지 돌파, 장생, 법보, 천겁, 자원 경쟁.
- 제목 신호: 수선, 선계, 도, 경지, 영근, 법보, 천겁.
- 1화에서 보여줄 것: 낮은 자질, 잔혹한 위계, 첫 기연 또는 선택.
- 5화 안에 증명할 것: 자원 획득, 경지 규칙, 대가.
- 반복 보상: 경지, 법보, 단약, 종문 지위, 비경 탐사.
- 인물 기능: 종문 문지기, 경쟁 제자, 사부, 자원 상인, 상위 경지 압박.
- 유료화 훅: 첫 경지 돌파 직후 더 큰 종문/비경/원한.
- 장기 확장: 하위 종문 -> 지역 패권 -> 상계/천도.
- 이탈 위험: 용어 과밀과 정서 거리감.
- 보정: 경지 설명은 생존 문제와 경쟁 장면 안에서 준다.

## 아카데미

- 구매 이유: 폐쇄된 경쟁 공간에서 재능, 지식, 관계로 올라간다.
- 제목 신호: 아카데미, 학원, 교관, 수석, 꼴찌, 반, 시험.
- 1화에서 보여줄 것: 학교 규칙과 주인공의 낮은/이상한 위치.
- 5화 안에 증명할 것: 첫 수업/시험/대결에서 특수성 증명.
- 반복 보상: 시험 성적, 반응, 파벌, 동아리, 관계 변화.
- 인물 기능: 라이벌, 교수, 문지기, 조력자, 감정축, 관찰자.
- 유료화 훅: 첫 공개 평가 직후 더 큰 시험이나 파벌 충돌.
- 장기 확장: 반/학년 -> 학교 전체 -> 외부 조직/전쟁.
- 이탈 위험: 학교가 배경만 되고 규칙이 없음.
- 보정: 시험, 규정, 평판이 실제 선택을 강제하게 한다.

## 로맨스판타지 / 악녀

- 구매 이유: 위계 사회에서 지위와 관계를 뒤집는다.
- 제목 신호: 악녀, 공작, 황태자, 파혼, 계약, 가문, 성녀, 회귀.
- 1화에서 보여줄 것: 사회적 위험, 평판, 파멸 플래그.
- 5화 안에 증명할 것: 첫 평판 반전, 관계 재배치, 생존 전략.
- 반복 보상: 사교 승리, 계약, 후회, 보호, 권력 균형 변화.
- 인물 기능: 남주/상대역, 가족 적대자, 궁정 문지기, 정보 조력자, 평판 관객.
- 유료화 훅: 첫 사교전 승리 직후 더 큰 스캔들/계약.
- 장기 확장: 개인 생존 -> 가문/궁정 권력 -> 왕국/전쟁.
- 이탈 위험: 로맨스와 정치가 따로 논다.
- 보정: 감정 선택이 권력 사건을 바꾸게 한다.

## 현대로맨스 / 오피스

- 구매 이유: 현실적인 관계 긴장과 감정 선택.
- 제목 신호: 상사, 계약, 이별, 재회, 사내연애, 비밀, 결혼.
- 1화에서 보여줄 것: 관계의 결핍과 직장/생활 압박.
- 5화 안에 증명할 것: 두 인물의 욕망과 저항 이유.
- 반복 보상: 오해, 협업, 거리 변화, 공개 위험, 감정 진전.
- 인물 기능: 상대역, 직장 문지기, 친구 조력자, 과거 인물, 관찰자.
- 유료화 훅: 관계가 사적으로만이 아니라 사회적으로 문제 되는 순간.
- 장기 확장: 개인 감정 -> 일/가족/사회적 선택.
- 이탈 위험: 사건 없이 감정만 반복.
- 보정: 프로젝트, 가족, 평판, 계약 같은 외부 사건을 관계에 묶는다.

## 비엘 / 지엘

- 구매 이유: 관계 긴장, 정체성, 욕망, 선택의 몰입.
- 제목 신호: 관계 키워드, 직업/무대, 계약, 가이드, 오메가, 재회, 혐관.
- 1화에서 보여줄 것: 두 인물의 거리감과 끌림/저항의 이유.
- 5화 안에 증명할 것: 관계가 사건을 움직인다는 증거.
- 반복 보상: 거리 변화, 질투, 보호, 오해, 선택, 공개 위험.
- 인물 기능: 중심 상대, 경쟁 관계, 외부 압박, 친구/조력자, 관찰자.
- 유료화 훅: 관계 정의가 흔들리거나 비밀이 외부로 새는 순간.
- 장기 확장: 사적 관계 -> 사회적 선택 -> 신뢰의 재정의.
- 이탈 위험: 한쪽 인물이 도구처럼 소비됨.
- 보정: 양쪽 욕망과 손실을 모두 설계한다.

## 아이돌 / 엔터

- 구매 이유: 무대, 팬덤, 성장, 여론 반응.
- 제목 신호: 아이돌, 데뷔, 연습생, 회귀, 센터, 무대, 팬덤.
- 1화에서 보여줄 것: 실패/결핍, 무대 욕망, 기회.
- 5화 안에 증명할 것: 첫 무대 또는 실력 증명.
- 반복 보상: 연습, 평가, 무대, 투표, 조회수, 팬덤 반응.
- 인물 기능: 멤버, 라이벌, 매니저, 프로듀서, 팬/안티 관객.
- 유료화 훅: 첫 공개 무대 성공 직후 더 큰 평가/스캔들.
- 장기 확장: 데뷔 -> 팬덤 확대 -> 업계 권력/글로벌 무대.
- 이탈 위험: 팬 반응 작위성, 멤버 장식화.
- 보정: 멤버별 역할과 팬덤 반응의 구체적 이유를 둔다.

## 스포츠

- 구매 이유: 훈련과 경기에서 기록과 인정을 얻는다.
- 제목 신호: 종목, 포지션, 천재, 회귀, 유망주, 기록.
- 1화에서 보여줄 것: 현재 실력/결핍, 경기 목표, 첫 검증.
- 5화 안에 증명할 것: 한 가지 기술 또는 판단의 차이.
- 반복 보상: 훈련, 경기, 기록, 팀 신뢰, 스카우트/팬 반응.
- 인물 기능: 코치, 라이벌, 팀원, 가족, 언론, 구단/협회.
- 유료화 훅: 첫 경기 성과 직후 더 큰 리그/상대.
- 장기 확장: 지역/학교 -> 프로/국가대표 -> 세계 무대.
- 이탈 위험: 경기 반복과 훈련 요약.
- 보정: 매 경기 다른 약점, 전술, 관계 압박을 둔다.

## 전문직 / 의학 / 법정

- 구매 이유: 전문성으로 문제를 해결하고 인정받는다.
- 제목 신호: 의사, 변호사, 검사, 판사, 셰프, 천재, 회귀, 사건.
- 1화에서 보여줄 것: 직업 현장의 압박과 주인공 판단.
- 5화 안에 증명할 것: 첫 사건 해결과 업계 반응.
- 반복 보상: 의뢰, 판단, 결과, 평판, 더 큰 사건.
- 인물 기능: 고객/환자, 상급자, 경쟁자, 기관 문지기, 현장 조력자.
- 유료화 훅: 첫 사건 성공 직후 더 어렵고 공개적인 사건.
- 장기 확장: 개인 의뢰 -> 조직/업계 -> 사회적 사건.
- 이탈 위험: 강의화.
- 보정: 전문 지식은 마감, 생명, 돈, 평판이 걸린 장면에서만 설명한다.

## 공포 / 괴담 / 미스터리

- 구매 이유: 이상한 규칙을 이해하고 살아남는다.
- 제목 신호: 금기, 규칙, 괴담, 저주, 사건, 탐정, 생존.
- 1화에서 보여줄 것: 일상 균열, 첫 금기, 첫 피해/위험.
- 5화 안에 증명할 것: 작은 규칙 하나와 더 큰 질문.
- 반복 보상: 단서, 규칙 이해, 생존, 반전, 금기 회피.
- 인물 기능: 피해자, 목격자, 조사자, 거짓 정보원, 위협자.
- 유료화 훅: 첫 규칙을 이해한 순간 더 큰 규칙이 드러남.
- 장기 확장: 개별 괴담 -> 연결된 원인 -> 최종 진실.
- 이탈 위험: 답을 너무 안 주거나 규칙이 임의적.
- 보정: 작은 답은 자주 주고 큰 질문은 천천히 키운다.

## 아포칼립스 / 생존 / 거점

- 구매 이유: 재난 속 생존, 자원 확보, 공동체 건설.
- 제목 신호: 종말, 아포칼립스, 생존, 쉘터, 거점, 재난, 좀비.
- 1화에서 보여줄 것: 재난의 즉시성, 주인공의 생존 선택.
- 5화 안에 증명할 것: 첫 자원 확보와 첫 생존 규칙.
- 반복 보상: 식량, 안전지대, 동료, 방어, 운영 규칙.
- 인물 기능: 가족/동료, 약탈자, 내부 갈등자, 기술자, 외부 집단.
- 유료화 훅: 첫 안전 확보 직후 더 큰 집단/재난.
- 장기 확장: 개인 생존 -> 거점 운영 -> 도시/국가 재건.
- 이탈 위험: 고통 누적과 운영 설명 과밀.
- 보정: 작은 안정 보상과 사이다를 주기적으로 넣는다.

## SF / 가상현실 / 사이버펑크

- 구매 이유: 낯선 기술/세계에서 정체성과 권력 문제를 해결한다.
- 제목 신호: AI, 가상현실, 우주, 함선, 기업, 사이버, 데이터.
- 1화에서 보여줄 것: 기술 규칙이 주인공 삶을 압박하는 장면.
- 5화 안에 증명할 것: 기술 우위 또는 시스템 균열.
- 반복 보상: 해킹, 장비, 정보, 구역 접근, 기업/권력 반응.
- 인물 기능: 기업 적대자, 기술 조력자, AI/시스템, 현장 동료.
- 유료화 훅: 작은 기술 문제 해결 직후 세계 구조의 더 큰 균열.
- 장기 확장: 개인 해킹/생존 -> 도시/기업 -> 인류/정체성 문제.
- 이탈 위험: 설정 설명 과밀과 감정 몰입 부족.
- 보정: 기술 설명은 손실, 관계, 생존, 선택에 연결한다.
